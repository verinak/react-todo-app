A simple To-Do application built with React that allows you to manage your tasks. All data is stored locally in the browser.

View live demo [here](https://verinak.github.io/react-todo-app).

## Features

- Create new tasks
- Mark tasks as completed
- Delete tasks
- Filter tasks by: search query, priority level, completion status, due date.
- Data persistence using LocalStorage
- User profile with 2D icons
- View/Edit task details page

## Technologies Used

- React
- JavaScript
- HTML5
- Tailwind CSS
