# React To-Do App

A simple To-Do application built with React that allows you to manage your tasks. All data is stored locally in the browser using LocalStorage.

View live demo [here](https://verinak.github.io/react-todo-app).

## Features

- Add new tasks
- Mark tasks as completed
- Delete tasks
- Task filtering
- Data persistence using LocalStorage
- User profile
- Task details page

## Technologies Used

- React
- JavaScript
- HTML5
- Tailwind CSS
